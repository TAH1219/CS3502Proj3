# CS3502Proj3
OwlTech File Manager

Should perform basic operations of a file explorer:

Basic operations:
Create
Read
Update
Delete
Rename
Navigate

Structure:
Form1.cs (Main)
Form1.Designer.cs
FileManager.cs

Design:
UI Layer handles all user interaction
File System Layer handles all direct calls to system.io

To build and run:
Tested on Windows. Use Visual Studio with .NET desktop development / Windows Forms
Project to be selected would be "CS3502Proj3.sln"
User should ensure it is set to WinForms
Build -> Build Solution
Run: Debug or start without debugging.
The application should start in the user's "Documents" folder.
